# UTAU用プラグイン even_scale

こちらは歌声合成ツール-UTAUにて使用可能なプラグインです。
選択範囲のノートを任意のスケール(音階)にスナップさせることができます。

# インストール・アンインストール

<インストール>
* GitHubからの場合：右上緑色の "Code" ボタンからZipファイルをダウンロード・解凍し、releaseフォルダ内にあるeven_scaleフォルダをそれごとご自分の "\UTAU\plugins" へコピーまたは移動させてください。
* Google Driveからの場合：リンク先のZipファイルをダウンロード・解凍し、even_scaleフォルダをそれごとご自分の "\UTAU\plugins" へコピーまたは移動させてください。

<アンインストール>
* アンインストールの際にはeven_scaleフォルダを削除することで完了します。

# 使用方法

UTAUからプラグインを起動し "↓CHOOSE SCALE THAT YOU WANT↓" 直下のピックリストから、お好みのスケールを選択してください。
その後 "Run" ボタンを押下するとプラグインが終了し、UTAUへノートの変更が反映されます。
ピックリストでなにも選択していない状態で "Run" ボタンを押下しても、安全にUTAUでの作業に戻ることができます。
